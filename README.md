# TuneTrek Frontend

Modern music streaming web application built with React, Vite, and Tailwind CSS.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables in .env

3. Start development server:
```bash
npm run dev
```

## Build

Build for production:
```bash
npm run build
```

## Deployment

Deploy to Vercel:
```bash
vercel --prod
```
